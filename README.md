## 使用技術

### フロントエンド
- Typescript
- React
- Chakra UI
- Vite

### バックエンド
- Python
- FastAPI

### データベース
- MySQL

### 開発ツール
- Docker


## 機能
- ユーザー認証機能
  - ユーザー登録
  - ログイン機能
- 研究室が所持する物品の表示機能
- 物品の貸出・返却機能
